WordPress plugin that adds categories and tags to pages.

== Description ==

This WordPress Plugin adds tags and categories to pages, just like posts have. 

* No setup required!

== Installation ==

Upload the "add-tags-categories-to-pages" folder to your wp-content/plugins directory, then Activate it.
